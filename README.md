# IOT SHOP (Exam)

## Get started

1. clone this repository.
2. install dependencies with this command below.

```bash
pnpm install
```

3. generate prisma.

```bash
npx prisma generate
```

4. create .env by copy from .env.example

```bash
cp .env.example .env
```

5. build nextjs

```bash
pnpm build
```

6. run docker compose

```bash
make up
```

7. run server

```bash
pnpm start
```

open http://localhost:3000
